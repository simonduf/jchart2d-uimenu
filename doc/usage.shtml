Usage

This page provides a quick howto for JChart2D. Additional information may be found at the documentation page.

Dependencies (3rd party jars)

Basically only the jchart2d-[version].jar
More jar files are only required if you use the Chart2D wrapped in the ChartPanel class and want to save EPS / PDF documents. You may use the ChartPanel (menu decorations) and not use the additional jar files. Then the menu to save EPS/PDF will be greyed out.

Save as EPS support

For EPS support you additionally need xmlgraphics-commons-[version].jar

linux: "java -cp jchart2d-3.3.0.jar:xmlgraphics-commons-1.5.jar: info.monitorenter.gui.chart.demos.AdvancedDynamicChart" Windows: "java -cp jchart2d-3.3.0.jar;xmlgraphics-commons-1.5.jar; info.monitorenter.gui.chart.demos.AdvancedDynamicChart"

Save as PDF support

For PDF support you additionally need

  • xmlgraphics-commons-[version].jar
  • fop-[version].jar
  • commons-logging-[version].jar
  • commons-io-[version].jar
  • batik-all-[version].jar
  • avalon-framework-[version].jar
